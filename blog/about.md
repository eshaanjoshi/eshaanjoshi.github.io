# About Me
## Who is this kid, anyhow?

### Introductions

Hi! My name is Eshaan. I'm an electrical engineer and a computer scientist. My big interests are in embedded systems, discrete mathematics, constructive logics, and whatever else has caught my eye that day. 

### What's this site?

I realized a long time ago that stuffing whatever I'd learned into the nearest composition notebook, college ruled, 80 pages, just wasn't cutting it. Instead, I'm putting it somewhere easier to reference to, and harder to lose. And also hopelessly overengineered. There are two bash scripts I run locally just to get this to generate the files for the web page. (It's markdown -> html and it's nonsense all the way down. I could have used Jekyll or something of that nature. Could have.)

Regardless, this is a semi-permanent way to record everything I did, what I was thinking while I did it, and why I was doing it. 

### Site Direction

The general idea is there's a list of all the blogs, and that's about as organized as I want to go. I'm trying to figure out an org-mode nodes style page where you can mouse around, but that might make more sense when and if I have more thoughts. Not ever blog will be earth-shattering, not every one will contain some new insight. 

### The name?

Gradient Descent is a machine learning algorithm in which you pick a random point, then take small steps from that point to the minimum error, in the direction of the steepest descent. Now, if machines can be prodded into learning, so can I, so here I'll be taking small steps towards something or the other, every day. See! It's a clever pun.
